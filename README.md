# Kidist Teshome — Designer Portfolio

Front-End & UI/UX designer portfolio built with **React**, **Vite**, and **Framer Motion**.

Live demo: _deploy and add your URL here_

## Features

- Responsive, feminine creative design
- Interactive **Vibe Coder** coding playground
- **Blush UI Playground** — mini design-system showcase
- GitHub & GitLab project links
- Smooth scroll animations & micro-interactions

## Showcase project

**[Blush UI Playground](docs/blush-ui-playground.md)** — a small interactive UI kit with 4 themes and live component preview. Perfect as a GitHub showcase repo.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```

## Push to GitHub

```bash
git init
git add .
git commit -m "Initial commit: designer portfolio with Blush UI showcase"
git branch -M main
git remote add origin https://github.com/kidistteshome27/portfolio.git
git push -u origin main
```

> Create the `portfolio` repository on [github.com/kidistteshome27](https://github.com/kidistteshome27) first if it doesn't exist.
